# Snub-yeah Security Site

- Dynamically updated security posts

### Installation
Clone the repository with
```
https://github.com/snub-yeah/Security-Site.git
```
Change into folder
```
cd Security-Site
```
Install dependencies
```
npm install
```
Start the developmental server
```
npm run dev
```
